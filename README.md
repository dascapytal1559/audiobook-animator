# Audibler

## Pipeline

starting with an audiobook
> split into chapters [reworked]
> split into chunks [reworked]
> transcribe chunks
> create scenes
> create image
> animate image
> stitch into video


## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file in the root directory with your API keys:
```
OPENAI_API_KEY=
DISCORD_TOKEN=
DISCORD_SERVER_ID=
DISCORD_CHANNEL_ID=
```
